## 🚀 快速开始

### 第一步：启动后端API
```powershell
cd backend
python app.py
```

### 第二步：启动前端应用
```powershell
cd metroplan
yarn install    # 首次需要安装依赖
yarn serve      # 启动开发服务器
```

### 第三步：打开浏览器
访问 `http://localhost:8080`，即可使用应用

## 📝 使用说明

1. 在"起点站"输入框输入车站名称（如：`广州南`）
2. 在"终点站"输入框输入车站名称（如：`深圳北`）
3. 点击"查询路径"按钮
4. 等待结果显示
5. 点击"获取车站列表"查看所有可用车站

## 🔌 API集成

### 调用的API接口：

1. **POST /path** - 查询路径
   - 请求：起点站、终点站、最大换乘次数
   - 响应：路径列表、时间、列车序列等

2. **GET /stations** - 获取车站列表
   - 响应：所有可用车站名称（65个）

## 🚨 常见问题

### Q：提示"网络错误"
**A：** 确保后端API已启动（`python app.py`）

### Q：查询没有结果
**A：** 检查车站名称是否正确，点击"获取车站列表"验证

### Q：样式显示不对
**A：** 确保执行了 `yarn install` 并且已重启开发服务器
